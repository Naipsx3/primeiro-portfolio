# Primeiro Portfólio

Este é o primeiro projeto para praticar programação web.

## O que você vai aprender aqui

- `index.html`: estrutura da página.
- `styles.css`: aparência, cores, espaçamentos e responsividade.
- `script.js`: interação com JavaScript.
- Git: salvar versões do projeto.
- GitHub: publicar o código online.

## Como abrir

Abra esta pasta no VS Code e use a extensão Live Preview no arquivo
`index.html`.

Também dá para abrir o arquivo `index.html` diretamente no navegador.

## Próximos passos

1. Trocar os textos para ficarem com a sua cara.
2. Adicionar seus links reais de contato.
3. Publicar o projeto no GitHub.
4. Ativar o GitHub Pages.

## Aula 1

Nesta etapa adicionamos uma seção de trilha de estudos e melhoramos o botão de
tema. O JavaScript agora usa `localStorage`, que salva uma pequena informação no
navegador para lembrar a escolha do usuário.

## Aula 2

Nesta etapa adicionamos links externos para o GitHub e para o projeto publicado.
No HTML, usamos `target="_blank"` para abrir em nova aba e `rel="noreferrer"`
para evitar enviar informação da página de origem ao site aberto.
